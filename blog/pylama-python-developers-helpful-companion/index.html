Pylama - Python developer's helpful companion

Posted: 09 3 2014 22:14 [Source]
Tags:  linter pep8 python

Everyone might have a hard time to keep his code clean, shiny according to pep8, and not to mention that there are other extensions to ` pep8 <http://legacy.python.org/dev/peps/pep-0008/>`_ rules, like pyflakes, or other linters for that matter. Of course, each linter can be installed, but imagine the need to run each separately, to determine what's in your code that should be fixed. Here comes Pylama.

Pylama is a tool combining power of other linters. It actually uses them beneath, wrapping in single config, command line tool and provides pre commit hooks for both git and mercurial to keep you from committing code, that violates your established style guide rules.

The linters that can be used are, as per Pylama documentation:

  • PEP8

  • PEP257

  • PyFlakes

  • Mccabe

  • Pylint

And additionally gjslint for checking your javascript style as well. (You need to install pylama_gjslint)
