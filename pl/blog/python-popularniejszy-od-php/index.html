Python popularniejszy od php

Opublikowano: 17 2 2011 21:02 [Źródło]
Tagi:  php python

Według indeksu TIOBE na Luty, python stał się językiem popularniejszym, niż php. na domiar złego, c# również depcze php po piętach.

Popularność C# można przynajmniej częściowo tłumaczyć ciągłym rozwojem platformy Mono - w C# powstał między innymi banshee, multimedialny kombajn, który zastąpi Rhythmboxa w kolejnych wydaniach Ubuntu.

Mnie osobiście z jednej strony cieszy rosnaca popularność pythona - od paru miesięcy jest moim głównym językiem, ale z drugiej martwi spadająca popularność php.
